Introduction
============

Onion is the web client for Ascribe. The idea is to have a well documented,
easy to test, easy to hack, JavaScript application.

The code is JavaScript ECMA 6.


Getting started
===============

```bash
git clone git@bitbucket.org:ascribe/onion.git
cd onion
npm install
npm run build
```
